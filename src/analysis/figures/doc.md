# Figures Directory

This directory contains all visual outputs generated by the analysis pipeline.

## Main Output

- **`dashboard.html`** — The interactive scrolling dashboard. This is the primary deliverable. Open it in any modern browser to explore all 7 charts with hover tooltips, click-to-drill sunbursts, and scroll-triggered animations. The file is fully self-contained (all chart data is inlined as JSON; only Plotly.js is loaded from a CDN).

## Regenerating

To rebuild the dashboard:

```bash
python src/analysis/09_dashboard.py
```

The PNGs are not regenerated by this script — they were produced by earlier notebooks that have since been removed from the repository.
